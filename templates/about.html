Maternal Health Prediction

Project Overview

Health Predictor is an application designed to assess health risks based on user-provided medical data. Leveraging machine learning, it predicts risk levels (Low, Medium, High) and provides tailored recommendations, empowering users to monitor their health effectively.

Process

  1. User Registration: New users sign up with their name, email, password, and role (User or Admin).
  2. Authentication: Users log in with their credentials, redirecting to their respective dashboards (User or Admin).
  3. Data Entry: Users input health metrics (e.g., age, blood pressure, blood sugar) via the "Enter New Data" page.
  4. Prediction: The system processes the data using a trained machine learning model to predict risk levels.
  5. Result Display: Predictions are shown on a result page, with an option to save them to the database.
  6. Dashboard Access: Saved records are displayed on the user dashboard, while admins view all users' latest records.

Workflow

The workflow is designed for simplicity and efficiency:

  • Welcome Page: Users start here and proceed to login.
  • Login/Signup: Secure authentication routes users to their dashboards.
  • User Dashboard: Displays the user's latest prediction and historical records, with an option to enter new data.
  • Data Input → Prediction → Save: Users enter data, view predictions, and save them, looping back to the dashboard.
  • Admin Dashboard: Shows the latest 10 records across all users for monitoring purposes. And the admin/doctor can give recommendation to the particular user.
  • Logout: Ends the session and returns to the welcome page.
